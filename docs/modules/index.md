# Modules Overview

Welcome to the modules section of the Physical AI & Humanoid Robotics book. This section contains all the core modules that will help you understand and implement Physical AI concepts.

## Available Modules

- **[Module 1: ROS 2 Fundamentals](/docs/modules/module-1-ros2/index)** - Learn the fundamentals of ROS 2 (Robot Operating System 2)
- **[Module 2: Simulation Environments](/docs/modules/module-2-simulation/index)** - Master simulation environments for safe robot development
- **[Module 3: NVIDIA Isaac Platform](/docs/modules/module-3-nvidia-isaac/index)** - Explore advanced tools from the NVIDIA Isaac platform
- **[Module 4: Vision-Language-Action Systems](/docs/modules/module-4-vla/index)** - Implement Vision-Language-Action (VLA) systems for human-robot interaction

Each module includes theoretical foundations, practical exercises, and real-world applications to provide you with comprehensive knowledge of Physical AI and embodied intelligence.
