# API Contracts

This directory contains the API contracts for the Physical AI & Humanoid Robotics book platform.

## Available Contracts

- `learning-outcomes-api.yaml`: OpenAPI specification for the learning outcomes management API

## API Documentation

All APIs follow RESTful principles and use JSON for request/response payloads. Authentication is handled via JWT tokens for secured endpoints.

## Versioning

APIs are versioned using the URL path (e.g., `/v1/`) to ensure backward compatibility during evolution.
